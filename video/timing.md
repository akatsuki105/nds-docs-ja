# 描画サイクルについて

NDSの画面描画は、2つの2Dエンジンがそれぞれの画面に対して次のように行います。

ドットクロック<sup>[1](#dot)</sup> = 5.585664 MHz (=33.513982 MHz / 6)

つまり、1ピクセル(=dot)描画するのに6クロックサイクルかかります。

```
      0                                           256           355
      ┌────────────────────────────────────────────┬─────────────┐
      │                                            │             │
      │                                            │             │
      │                                            │             │
      │                                            │             │
      │                    Screen                  │   HBlank    │
      │                                            │             │
      │                                            │             │
      │                                            │             │
  192 ├────────────────────────────────────────────┴─────────────┤
      │                                                          │
      │                        VBlank                            │
      │                                                          │
  263 └──────────────────────────────────────────────────────────┘
```

```
  HBlank: 256~355 までの 99 ドット (15.7343KHz)
  VBlank: 192~263 までの 71 ライン (59.8261 Hz)
    3Dエンジンの VBlank サイクルは 191~213 の 23 ライン
```

## 注釈

<sup id="dot">1: ドット(dot)とは、PPUが1つのピクセルを出力できる最短期間のことです。</sup>
